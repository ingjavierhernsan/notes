- Como son esos procesos.
	Registro y modificación de entrenamientos nuevos
		Cuando un manufacturing manager tiene la necesidad de entrenar a su personal con algún curso nuevo debe comunicarse con el personal de entrenamiento de forma personal o correo electrónico. Se plantea de manera formal la necesidad del curso. En base a la experiencia del personal de entrenamiento se solicita información sobre el curso en cuestión al manufacturing manager, esta información se plantea en el documento xxxx.docs. Se revisa si de forma interna se puede dar el curso, si se tienen los recursos, si esto no es posible se consideran las opciones externas y con la información completa se pasa a un siguiente nivel para su aprobación. Cuando se aprueba el curso la información de este se captura en el documento de Microsoft Excel de matriz de entrenamiento.
		
		También se puede tener cursos nuevos que ocupan impartir por parte de otras áreas de la empresa a los empleados de otras áreas, un ejemplo es Seguridad e higiene. Con la información de este curso nuevo se captura en el documento de Microsoft Excel de matriz de entrenamiento.
			
	Registro y modificación de cursos a empleados
		Este registro se hace en un documento de Microsoft Excel. Este documento se le denomina matriz de entrenamiento, cada área en la empresa tiene un documento donde se registra que cursos debe tener el puesto. En el documento se tienen varias pestañas, en la pestaña Catalogo se registran todos los cursos, se tiene una pestaña por cada puesto del área y se registran los cursos que debe tener ese puesto.
		
		Ejemplo del documento de matriz de entrenamiento del departamento de IT: IT Matrix.xlsx
		
		También se tiene un documento de Microsoft Excel llamado Training Records_final 2.002.xlsx donde se registra los entrenamientos que han realizado cada empleado.
	Planeación de cursos
		-Internos
			-Por parte del are de entrenamiento
				-Nuevo ingreso
					Se revisan los lunes el documento de Microsoft Excel New Hire 2023.xlsx que edita el área de recursos humanos, aquí se indican que personas son de nuevo ingreso y que fecha será su ingreso a la empresa como empleados. En base a su fecha de ingreso se prepara el curso de inducción para estos empleados.
					
					Recursos humanos se encarga de indicarle a los empleados de nuevo ingreso en donde se darán los cursos de inducción.
				-Empleados normales
					Se revisa en los documentos de Microsoft Excel que empleados necesitan algún curso para terminar su capacitación. Cuando se encuentra algún curso faltante se revisa en unas listas de cursos asignados a las áreas. Esto sirve para formar un grupo de empleados y aprovechar ese tiempo para capacitar a más empleados. 
			-Por parte de otra área
				Se encarga de desarrollar el curso que impartirá a empleados de otras áreas. Estos cursos deben ser proporcionados al área de entrenamiento para su captura en la matriz de entrenamiento. 
				
				El área que dará el entrenamiento debe informarle al área de entrenamiento que tiene que impartir el curso, además de solicitarle algún espacio para este mismo.
				
				Al finalizar el curso, el área debe dar la lista de asistencia de los empleados al área de entrenamiento para agregar este entrenamiento a los registros del empleado.
		-Externos
			El área de entrenamiento se encarga de capturar en la matriz de entrenamiento el curso. Además, se encarga de preparar el área donde se dará el curso en las instalaciones de la empresa, convocara a los empleados que recibirán el curso. El encargado del curso externo proporciona la lista de asistencia al curso para que el área de entrenamiento agregue el entrenamiento a los registros del empleado.
	Registro de asistencia a cursos
		Por cada curso que se imparte se debe llevar un registro de los empleados que tomaron el curso y si lo terminaron. El área de entrenamiento registra la asistencia de los empleados a los cursos en una hoja de registro denominada REGISTRO DE ENTRENAMIENTO y posteriormente se captura en un documento de Microsoft Excel.
	Registro de resultados de los cursos
		No se registran los resultados del curso impartido. Solo se tiene un estatus de este si no se terminó de impartir.
	Impartición de cursos a empleados
		-Dados por el área de entrenamiento.
			-Cursos de inducción
				Los cursos que se impartirán dependerán a que área y puesto serán asignados los empleados, en base a esto se imprime el documento Hoja de Verificacion - Induccion al Personal, este documento se lleva el control de que cursos se han impartido al empleado.			
			-Cursos necesarios por área
				Cuando ya se tiene formado el grupo se asigna una fecha para impartir el curso, se le informa al supervisor del empleado para que se coordine con el empleado.
	Registro y expedición de certificado de cursos impartidos registrados en gobierno
		Si el curso se encuentra dentro del catálogo del formato DC-3 se llena el formato que se tiene capturado en un documento de Microsoft Word, se imprime y se archiva en los registros del empleado. Si el empleado solicita el certificado, se imprime y se entrega.
